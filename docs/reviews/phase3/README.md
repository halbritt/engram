# Phase 3 Reviews

This directory stores Phase 3 specification, build-prompt, implementation, and
pipeline review artifacts.

The marker files in `markers/` are used by tmux-run agents to coordinate
handoffs on the same branch. See
`docs/process/phase-3-agent-runbook.md` for marker names and required content.

Expected artifact families:

- `PHASE_3_CLAIMS_BELIEFS_SPEC_REVIEW_<model>_2026_05_05.md`
- `PHASE_3_CLAIMS_BELIEFS_SPEC_FINDINGS_LEDGER_2026_05_05.md`
- `PHASE_3_CLAIMS_BELIEFS_SPEC_SYNTHESIS_2026_05_05.md`
- `PHASE_3_CLAIMS_BELIEFS_SPEC_SYNTHESIS_REVIEW_<model>_2026_05_05.md`
- `PHASE_3_BUILD_PROMPT_REVIEW_<model>_2026_05_05.md`
