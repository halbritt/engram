# Quarantine Notice

This directory is preserved as RFC 0032 audit evidence.

The artifacts in this directory are not authoritative review evidence.
RFC 0032 found that the advertised independent review lanes were not
backed by independent model-lane output: the Striatum subprocesses
failed to produce usable lane artifacts, and the review files were filled
through Striatum's recovery path.

Do not use `REVIEW_*.md`, `FINAL_REVIEW.md`, `FINDINGS_LEDGER.md`,
`RUN_SUMMARY.md`, or handoff files in this directory as acceptance
evidence for RFC 0028. See
`../rfc0032-suspect-work-audit/PROVENANCE_AUDIT.md` and
`../rfc0032-suspect-work-audit/ARTIFACT_DISPOSITION.md`.
