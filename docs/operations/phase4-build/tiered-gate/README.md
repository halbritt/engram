# Phase 4 Tiered Gate Operations

This directory holds aggregate-only RFC 0024 Tier 0-2 gate reports for Phase 4.

The active scaffold is:

```text
striatum/phase-4-tiered-gate/workflow.json
```

This workstream intentionally skips human involvement. Missing human-labeled
entity and review evidence must be recorded as `deferred_until_rfc0021` and
revisited after the RFC 0021 interview loop lands.

Reports in this directory must not include raw corpus text, belief values,
claim values, entity names, prompts, completions, conversation titles, or
home-directory absolute paths.
