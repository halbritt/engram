# Striatum Memory RFC Alignment Workflow

This workflow follows the completed Striatum memory roadmap RFC execution. It
is proposal/document alignment only.

Initial parallel lanes:

- RFC 0046 projection/index alignment
- RFC 0047 retrieval-boundary alignment
- RFC 0048 context/audit policy alignment
- RFC 0049 gate wording alignment
- roadmap/RFC index cleanup

After those lanes complete, run the independent review fan-out, findings
ledger, and final synthesis.

Do not use this workflow to promote RFCs or implement code.
