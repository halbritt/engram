# Review Prompt

Review the RFC alignment package produced by this workflow. You are reviewing
proposal text and operator documentation only; do not implement code or edit
source artifacts.

Use a fresh context and the maximum useful number of read-only sub-agents. Check
that the package remains local-only, proposal-only, non-promoting, and aligned
with the prior final synthesis.

Write only the expected review artifact. Include:

- verdict: `accept`, `accept_with_findings`, or `needs_revision`;
- blocker findings, if any;
- nonblocking findings and deferred work;
- affected files and evidence paths;
- any operator workflow friction.
