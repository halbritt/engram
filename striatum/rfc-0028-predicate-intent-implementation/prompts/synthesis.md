# RFC 0028 Revision Synthesis

Read `FINDINGS_LEDGER.md`, RFC 0028, and the implementation handoff. Produce an
actionable revision plan.

Write
`docs/reviews/rfc0028-predicate-intent-implementation/REVISION_SYNTHESIS.md`
with the exact lowercase `author:` line from the work packet.

For every ledger finding, classify it as `accepted`, `accepted_with_modification`,
`deferred`, or `rejected`, with a one-sentence reason. Then list concrete
revision tasks for the apply job. If no code change is needed, say so
explicitly.

Do not modify implementation files.
